# Variables

## Declarations

Defined with either the *let* or *var* operators.

```
let a = 1  // immutable
var b = 2  // mutable

let c: i32; // error(immutable declarations must be assigned at declaration time)


```
### Compound

```
var d, e, f: f64;
```

## Assignment

```
a = 1 // error(immutable)
b = 3 // b = 3
```
